# FaBo SLAM


## Requirements
* Jetson TX2 JetPack 4.2
* PC(optional)
* ROS Melodic
* Cartographer-ROS
* LIDAR (one of these)
  * YDLIDAR X4
  * Hokuyo UST-10LX
